OBAT CLASS HIERARCHY:

┌─ Obat.cs (TubesKPL)
│
├─ Properties (PascalCase - Primary)
│  ├─ Id                    [dari server]
│  ├─ Nama                  [sesuai PascalCase convention]
│  ├─ Kategori (string)     [untuk API compatibility]
│  ├─ Stok
│  ├─ Harga
│  ├─ ExpiredDate
│  └─ Status (string)       [untuk API compatibility]
│
├─ Alias Properties (camelCase - Backward Compat)
│  ├─ nama          → Nama       (get/set wrapper)
│  ├─ stok          → Stok       (get/set wrapper)
│  ├─ harga         → Harga      (get/set wrapper)
│  ├─ expiredDate   → ExpiredDate (get/set wrapper)
│  ├─ kategori      → Kategori   (get/set wrapper + enum conversion)
│  └─ status        → Status     (get/set wrapper + enum conversion)
│
├─ Helper Methods
│  ├─ UpdateStatus()       [update status berdasarkan stok & expired]
│  ├─ GetStatusEnum()      [convert string Status ke enum]
│  └─ ToString()           [untuk debug/logging]
│
├─ Multiple Constructors
│  ├─ new Obat()                                           [default]
│  ├─ new Obat(nama, stok, harga, expiredDate, kategori)  [string kategori]
│  └─ new Obat(nama, stok, harga, expiredDate, enum)      [enum kategori]
│
└─ Enums (untuk backward compat)
   ├─ StatusObat (Available, LowStock, Expired)
   └─ KategoriObat (Tablet, Salep, Sirup, Vitamin, Antibiotik, AntiJamur)
